# 🎮 Platform Disappear 2D
# 🧠 Sobre o jogo

Platform Disappear 2D é um jogo de sobrevivência em que o jogador precisa se manter vivo o máximo de tempo possível enquanto enfrenta plataformas instáveis e inimigos.

O objetivo é simples: não cair e não morrer.

# 🎯 Como jogar
Use A / D ou ← / → para se movimentar
Pressione Espaço para pular
Segure Shift para correr
Use o botão de ataque (tecla X) para eliminar inimigos

## Core experience:

Quero que o jogador sinta tensão enquanto tenta sobreviver o máximo possível, pulando entre plataformas instáveis, desviando de inimigos e usando habilidades no momento certo.

## ▶️ Como executar
1. Abra o jogo (versão WebGL ainda não está disponível)
2. Clique em Play
3. Comece a jogar
